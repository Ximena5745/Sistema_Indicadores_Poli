## 🔗 Cómo Acceder

1. **Abrir HTML**
   ```bash
   open dashboard_profesional_v2.html
   ```

2. **Navegar entre tabs**
   - Click en iconos del sidebar izquierdo

3. **Interactuar con gráficos**
   - Hover → Muestra tooltips
   - Click → Zoom (algunos gráficos)

4. **Aplicar filtros**
   - Seleccionar Año, Mes, Modalidad, Proceso
