# ADR-002: Frontend — Next.js 14

**Estado:** Aceptado  
**Fecha:** 2026-06-13

## Decisión

Next.js 14 con App Router, TypeScript, Tailwind CSS, React Query, Zustand.

## Alternativas

- React SPA (Vite) — menos SSR/SEO, sin convenciones de routing
- Mantener Streamlit — no cumple requisitos UX/performance

## Consecuencias

- Curva de aprendizaje para equipo Python
- shadcn/ui se integrará en Fase 3-5
