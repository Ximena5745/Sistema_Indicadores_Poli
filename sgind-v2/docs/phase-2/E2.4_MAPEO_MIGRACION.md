# E2.4 — Mapeo Migración SQLite → PostgreSQL

## Origen

| Parámetro | Valor |
|-----------|-------|
| Archivo | `data/db/registros_om.db` |
| Motor | SQLite 3 |
| Tablas | `registros_om`, `acciones` |

## `registros_om`

| SQLite | PostgreSQL | Transformación |
|--------|------------|----------------|
| id | — | No migrar (nuevo SERIAL) |
| id_indicador | id_indicador | str() |
| nombre_indicador | nombre_indicador | directo |
| proceso | proceso | directo |
| periodo | periodo | directo |
| anio | anio | int() |
| sede | sede | directo (nuevo en PG) |
| tiene_om | tiene_om | int 0/1 |
| — | tipo_accion | default 'OM Kawak' si ausente |
| numero_om | numero_om | directo |
| comentario | comentario | directo |
| registrado_por | registrado_por | directo |
| fecha_registro | fecha_registro | directo |

**Conflict resolution:** `ON CONFLICT ON CONSTRAINT registros_om_unique_key DO UPDATE`

## `acciones`

| SQLite | PostgreSQL | Transformación |
|--------|------------|----------------|
| id | — | Nuevo BIGSERIAL |
| ACCION | accion | rename |
| RESPONSABLE | responsable | rename |
| ESTADO | estado | rename |
| test_marker | marker_value | marker_col='test_marker' |
| (otras columnas) | payload | JSONB |

## Validación post-migración

```sql
-- Conteos
SELECT 'registros_om' AS tabla, COUNT(*) FROM registros_om
UNION ALL
SELECT 'acciones', COUNT(*) FROM acciones;

-- Duplicados OM (debe ser 0)
SELECT id_indicador, periodo, anio, COUNT(*)
FROM registros_om
GROUP BY 1,2,3 HAVING COUNT(*) > 1;

-- FK usuarios (sin huérfanos)
SELECT COUNT(*) FROM audit_log al
LEFT JOIN users u ON u.id = al.user_id
WHERE al.user_id IS NOT NULL AND u.id IS NULL;
```

## Rollback

```sql
TRUNCATE acciones RESTART IDENTITY CASCADE;
TRUNCATE registros_om RESTART IDENTITY CASCADE;
-- audit_log se limpia en cascada si hay FK, o:
DELETE FROM audit_log WHERE entity = 'registros_om';
```

## Script

```bash
python sgind-v2/database/scripts/migrate_sqlite_to_postgres.py \
  --sqlite data/db/registros_om.db \
  --postgres postgresql://sgind:sgind_dev_password@localhost:5432/sgind
```

Reporte generado en `sgind-v2/database/reports/migration_*.json`
