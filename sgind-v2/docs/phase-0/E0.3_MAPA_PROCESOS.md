## 4. Flujo de autenticación Microsoft OIDC

```mermaid
sequenceDiagram
    participant U as Usuario
    participant APP as streamlit_app
    participant GUARD as auth_modules/guards.py
    participant OIDC as Microsoft Entra ID
    participant SEC as st.secrets

    U->>APP: Accede URL app
    APP->>GUARD: require_auth()
    alt st.user.is_logged_in = false
        GUARD->>U: st.login(provider=microsoft)
        U->>OIDC: OAuth2/OIDC
        OIDC->>APP: email, name, email_verified
    end
    GUARD->>SEC: get_allowed_emails()
    alt email not in whitelist
        GUARD->>U: Acceso denegado
    else autorizado
        GUARD->>APP: user_info dict
        APP->>U: Render página solicitada
    end
```
