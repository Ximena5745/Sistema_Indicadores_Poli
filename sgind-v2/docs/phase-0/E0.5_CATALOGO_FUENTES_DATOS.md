## 6. Trazabilidad fuente → indicador → dashboard

```mermaid
flowchart LR
    API[API/Kawak Excel] --> CAP[consolidar_api.py]
    CAP --> CAK[Consolidado_API_Kawak.xlsx]
    CAK --> AC[actualizar_consolidado.py]
    FUENTE[Resultados_Consolidados_Fuente.xlsx] --> AC
    AC --> RC[Resultados Consolidados.xlsx]
    RC --> DL[data_loader.cargar_dataset]
    DL --> CALC[core/calculos.py]
    CALC --> DASH[Dashboard Streamlit]
    OM[(registros_om)] --> GOM[gestion_om.py]
```
